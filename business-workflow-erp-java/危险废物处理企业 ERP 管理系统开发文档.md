危险废物处理企业 ERP 管理系统 - 开发文档
## 项目基本信息
项目工期：2 个月（40 个工作日），预留 2 周（10 个工作日）缓冲期，应对需求调整、不可抗力等因素导致的工期紧张问题
## （二）项目报价
总报价：22,000 元
## （三）开发模式
敏捷开发，分阶段交付，每阶段完成后组织需求方验收，确保功能符合预期
## （四）技术栈
前端：Vue 3 + TypeScript + Element Plus + Vite + ECharts（可视化报表）
后端：Spring Boot + Spring Security + MyBatis-Plus + 微服务架构（可选扩展）
数据库：MySQL 8.0+ + Redis（缓存）
部署：服务器部署（支持云服务器 / 物理服务器）
## （五）维护说明
免费维护期：上线后前 3 个月，涵盖系统 Bug 修复、服务器基础维护（如重启、日志清理、环境配置优化）
付费维护期：
3个月内若需调整代码、修改需求（非 Bug 类），按 2,000 元 / 月收取服务费
后续系统升级（新增功能模块、架构扩展等）费用另行评估报价

# 三、业务分解
## （一）核心业务模块
客户关系管理模块
客户登记备案（手动录入 / 批量导入、资质上传、名录匹配）
客户报价管理（报价单生成、调整、导出）
客户跟进管理（跟进日志、提醒设置、进度跟踪）
合同管理模块
合同创建与审核（合规审核、扫描件归档）
合同执行跟踪（进度展示、全流程追溯）
合同完结归档（自动判断完结条件、电子档案存储）
生产执行管理模块
调度管理（收运通知接收、车辆 / 人员分配、运输单生成）
运输管理（系统不参与）
仓储管理（入库核对、过磅记录、库存更新）
财务管理模块
结算管理（结算单生成、审核、对账（电子回单匹配））
发票管理（发票录入（红字 蓝字）、扫描件上传、关联结算单）
基础数据与系统管理模块
基础数据维护（危险废物名录、批量导入导出）
权限与用户管理（三级权限配置、角色自定义、登录日志）
日志管理（操作日志、数据变更日志、异常日志）
报表与决策支持模块
业务报表（客户开发、合同执行、生产作业、库存报表）
财务报表（结算报表、账龄分析报表）
数据导出（Excel/PDF 格式导出）
